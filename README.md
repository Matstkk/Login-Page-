# 🔐 Modern Auth Interface - Login & Register

Este projeto é um módulo de autenticação desenvolvido para demonstrar o fluxo completo de entrada e registro de usuários, focando em uma interface limpa, responsiva e com validações em tempo real.

---

### 🌐 Demo Online
🚀 **Acesse o projeto em execução:** [https://matstkk.github.io/Login-Page-/](https://matstkk.github.io/Login-Page-/)

---

### 🚀 Funcionalidades
* **Autenticação Dinâmica:** Alternância fluida entre as telas de Login e Cadastro.
* **Validação de Formulários:** Verificação de campos (e-mail válido, força de senha e campos obrigatórios) antes do envio.
* **Segurança Visual:** Opção de "mostrar/esconder" senha para melhorar a experiência do usuário.
* **Persistência de Sessão:** Implementação de lógica para "Lembrar-me" (utilizando LocalStorage).
* **Design Responsivo:** Interface adaptável para qualquer dispositivo (Desktop, Tablet e Mobile).

### 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura semântica focada em acessibilidade.
* **CSS3 / Tailwind:** Estilização moderna com variáveis para fácil customização de temas.
* **JavaScript (ES6+):** Manipulação de eventos, lógica de validação e controle de transições.

### 🧠 Conceitos Aplicados
* **UX/UI Design:** Foco na redução da carga cognitiva durante o preenchimento dos dados.
* **Data Handling:** Tratamento e higienização de inputs para evitar erros de processamento.
* **State Management:** Controle de estados da interface (carregando, erro ou sucesso).
